## Overview

**PutraNET** is Putrajaya's dedicated fiber optic network infrastructure, complemented by a city-wide **LoRaWAN** (Long Range Wide Area Network) for IoT sensor connectivity, forming the backbone of smart city operations.

## PutraNET Fiber Network

### Infrastructure
- City-wide fiber optic backbone
- Network nodes at strategic locations
- High-capacity data transmission
- Redundant connectivity paths

### Applications
- CCTV video transmission
- Building management systems
- Traffic control systems
- Government network connectivity

## LoRaWAN Network

### Technology
- Low Power Wide Area Network (LPWAN)
- Long-range communication (up to 15km)
- Low power consumption
- Cost-effective sensor connectivity

### Features
- Fair/neutral host model
- Multi-tenant support
- City-wide coverage
- Secure data transmission
