## Overview

The **Centralized Electronic Bulletin Board (CEBB)** is a comprehensive digital signage network that enables Perbadanan Putrajaya to communicate effectively with residents, workers, and visitors. The system comprises LED billboards and smart digital displays strategically positioned throughout the city.

## System Architecture

### Display Infrastructure

The CEBB network consists of various display types:

| Type | Location | Purpose |
|------|----------|---------|
| Large LED Billboards | Major roads, intersections | High-visibility announcements |
| Digital Kiosks | Government buildings, public spaces | Interactive information |
| Indoor Displays | Lobbies, waiting areas | Service information |
| Transit Displays | Bus stops, pedestrian areas | Transport updates |

### Central Management System

All displays are controlled through a unified content management system (CMS):

- **Web-Based Interface**: Browser-accessible management dashboard
- **Role-Based Access**: Tiered permissions for content approval
- **Real-Time Updates**: Instant content deployment across network
- **Scheduling Engine**: Automated content rotation and timing
- **Analytics Dashboard**: Display performance and reach metrics

## Key Features

### Emergency Broadcasting

The CEBB serves as a critical emergency communication channel:

- **Instant Override**: Emergency content takes immediate priority
- **Multi-Location Sync**: Synchronized alerts across all displays
- **PCC Integration**: Direct connection to Command Centre
- **Pre-Programmed Templates**: Ready-to-deploy emergency messages

### Multi-Language Support

Content is displayed in multiple languages:
- Bahasa Malaysia
- English
- Mandarin
- Tamil

### Scheduling & Automation

Advanced scheduling capabilities include:
- Time-based content rotation
- Day-of-week programming
- Special event scheduling
- Seasonal content adjustments

### Remote Management

The entire network is managed remotely:
- Display health monitoring
- Brightness auto-adjustment
- Content verification
- Fault detection and alerts
