## Overview

The **Automatic Number Plate Recognition (ANPR)** system enhances Putrajaya's security infrastructure by automatically capturing and processing vehicle registration plates at strategic locations throughout the city.

## Technology

### Recognition System
- High-resolution cameras
- AI-powered plate recognition
- Multi-lane coverage
- Day and night operation
- High accuracy rates

### Data Processing
- Real-time plate extraction
- Database matching
- Alert generation
- Historical logging
- Analytics processing

## Applications

### Security
- Stolen vehicle detection
- Watchlist monitoring
- Criminal investigation support
- Access control verification
- Event security enhancement

### Traffic Management
- Vehicle flow counting
- Journey time analysis
- Traffic pattern studies
- Congestion monitoring
- Origin-destination surveys

## Strategic Locations

Deployed at:
- Main city entry points
- Highway interchanges
- Key intersections
- Government complex entrances
- Event venue approaches

## Integration

### Command Centre
- Real-time alert display

- Report generation

- Evidence collection
