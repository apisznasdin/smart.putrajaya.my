## Overview

**OSC 4.0** (One Stop Centre 4.0) is a comprehensive online submission system developed by Perbadanan Putrajaya for planning and building permit applications. This digital platform streamlines the development approval process, reducing paperwork, improving transparency, and accelerating decision-making.

## Platform Access

Access OSC 4.0 at: **[osc.ppj.gov.my](https://osc.ppj.gov.my/)**

## System Features

### Online Plan Submission

The platform enables complete digital submission:

| Feature | Description |
|---------|-------------|
| Document Upload | Submit plans and drawings digitally |
| Form Filling | Complete applications online |
| Checklist Validation | Ensure complete submissions |
| Fee Calculation | Automatic fee computation |
| Payment Gateway | Online payment processing |

### Application Tracking

Applicants can monitor their submissions:

- **Status Updates**: Real-time application progress
- **Stage Tracking**: View current processing stage
- **Comment Access**: Read technical comments
- **Timeline View**: See processing milestones
- **Notification Alerts**: Email/SMS updates

### Multi-Agency Coordination

OSC 4.0 facilitates coordination among:

- Perbadanan Putrajaya (Planning)
- Fire and Rescue Department
- Utility Providers (TNB, Water, Telco)
- Environmental Department
- Other Technical Agencies

## Process Flow

### Application Journey

```
1. Registration → Create OSC 4.0 account
2. Preparation → Compile documents
3. Submission → Upload and submit online
4. Processing → Multi-agency review
5. Comments → Technical feedback
6. Revision → Address comments (if any)
7. Approval → Digital approval issued
8. Payment → Online fee settlement
9. Collection → Download approved plans
```

### Timeline Standards

Perbadanan Putrajaya commits to processing timelines:

| Application Type | Target Duration |
|-----------------|-----------------|
| Planning Permission | 30-60 working days |
| Building Plan | 14-30 working days |
| Minor Applications | 7-14 working days |

*Timelines subject to complete submissions and no objections*

## Technical Specifications

### System Architecture

- **Platform**: Web-based application
- **Access**: Browser-compatible, mobile-responsive
- **Security**: SSL encryption, secure authentication
- **Integration**: Connected to agency systems
- **Availability**: 24/7 online access

### User Requirements

- Valid email address
- Company/Individual registration
- Digital copies of submission documents
- Payment method (online banking/card)

## Registration Process

### For Professionals

1. Visit [osc.ppj.gov.my](https://osc.ppj.gov.my/)
2. Click "Register"
3. Complete professional details
4. Upload qualification documents
5. Await verification
6. Receive login credentials

### For Landowners/Developers

1. Access the portal
2. Select appropriate user type
3. Provide identification details
4. Complete registration form
5. Verify email address
6. Start submitting applications
